# Privacy Policy

Godex is an independent open source Codex plugin. The plugin content in this repository does not collect, transmit, sell, or store user data by itself.

Godex provides local skill instructions and reference files for Codex agents. Any data access, logging, model usage, or network activity comes from the host Codex environment, tools the user chooses to run, or repositories the user asks Codex to inspect.

Do not include secrets, tokens, connection strings, private keys, personal data, or confidential production data in public issues, examples, prompts, logs, or benchmark artifacts.
