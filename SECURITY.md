# Security Policy

Godex is an independent open source Codex plugin. Do not report private service credentials, production data, or customer data in public issues.

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.0.x | Yes |

## Reporting a Vulnerability

Open a private advisory or contact the maintainer through the repository owner profile. Include:

- affected version or commit
- affected file or skill section
- reproduction steps using public, non-sensitive examples
- expected security impact

Godex guidance must not encourage logging secrets, tokens, connection strings, PII, raw payloads, or internal-only data.
